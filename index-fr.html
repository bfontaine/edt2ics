edt2ics

Convertissez les emplois du temps HTML du site de l’UFR Informatique de Paris Diderot vers des fichiers iCalendar.

Voir le projet sur GitHub bfontaine/edt2ics

  • Voir sur GitHub

edt2ics est une bibliothèque et outil en ligne de commande pour les étudiants en informatique à Paris Diderot pour générer des fichiers iCalendar depuis les emplois du temps en ligne

iCalendar est un format standard pour les données de calendrier, supporté par une multitude d’applications. Ce format vous permet d’avoir votre emploi du temps directement dans le calendrier de votre téléphone, par exemple.

Installation

pip install p7edt2ics

Utilisation

edt2ics [--semester <semester>] <year>

Où year est: L3, M1, ou M2 et semestre est 1 (par défaut) ou 2.

Les emplois du temps sont exportés tels qu’ils apparaissent sur le site, c’est à vous de supprimer les cours en trop dans votre application de calendrier.
